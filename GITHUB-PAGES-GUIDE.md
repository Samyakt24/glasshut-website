## 📋 Step-by-Step Instructions

### STEP 1: Create GitHub Account (2 minutes)
1. Go to https://github.com
2. Click "Sign up"
3. Enter your email, password, and username
4. Verify your email
5. Done!

### STEP 2: Create New Repository (1 minute)
1. Click the "+" icon (top right) → "New repository"
2. Repository name: `glass-hut` (or any name you like)
3. Description: "Glass Hut Website - Handcrafted Glass Bangles"
4. Make it **Public** (required for free hosting)
5. ✅ Check "Add a README file"
6. Click **"Create repository"**

### STEP 3: Upload Your Website Files (3 minutes)
1. You'll see your new repository page
2. Click "Add file" → "Upload files"
3. **Drag and drop ALL these files and folders:**
   - index.html
   - README.md
   - css/ (folder)
   - js/ (folder)
   - images/ (folder)
   - data/ (folder)
4. At the bottom, click **"Commit changes"**
5. Wait for upload to complete (green checkmark)

### STEP 4: Enable GitHub Pages (2 minutes)
1. Click "Settings" tab (top of repository)
2. Scroll down and click "Pages" (left sidebar)
3. Under "Source":
   - Branch: Select **"main"**
   - Folder: Select **"/ (root)"**
4. Click **"Save"**
5. Wait 2-3 minutes for deployment

### STEP 5: View Your Live Website! 🎉
1. Refresh the Settings → Pages page
2. You'll see a green box that says:
   **"Your site is live at https://yourusername.github.io/glass-hut/"**
3. Click the link to view your website!
4. Share this URL with anyone!

## 🎯 How to Update Your Website Later

### Adding New Products:
1. Go to your repository
2. Click on `data` → `products.json`
3. Click the pencil icon (Edit)
4. Add your new product entry
5. Click "Commit changes"
6. Your website updates automatically in 1-2 minutes!

### Adding New Images:
1. Go to your repository
2. Navigate to `images/products/`
3. Click "Add file" → "Upload files"
4. Upload your new product images
5. Update `products.json` with the new image paths
6. Done!

### Updating Styles or Content:
1. Navigate to the file you want to edit
2. Click the pencil icon
3. Make your changes
4. Click "Commit changes"
5. Website updates automatically!

## 💡 Pro Tips

### Tip 1: Always Preview Before Committing
- After editing files, preview your changes
- Make sure everything looks good

### Tip 2: Keep Image Sizes Small
- Large images = slow website
- Use https://tinypng.com to compress images
- Keep product images under 200KB each

### Tip 3: Test on Mobile
- Open your website on your phone
- Make sure it looks good on small screens
- GitHub Pages works perfectly on all devices!

### Tip 4: Regular Backups
- GitHub keeps version history
- You can always revert to previous versions
- Click "History" to see all changes

## 🆘 Common Issues & Solutions

### Issue: "404 Page Not Found"
**Solution:** 
- Wait 3-5 minutes after enabling Pages
- Check that index.html is in the root folder
- Refresh and clear browser cache

### Issue: Images Not Loading
**Solution:**
- Check image paths in products.json
- Make sure images are uploaded to images/products/
- Image names are case-sensitive (use lowercase)

### Issue: Website Not Updating
**Solution:**
- Wait 2-3 minutes after committing changes
- Clear browser cache (Ctrl + Shift + R)
- Check that changes were committed successfully

### Issue: CSS/JS Not Working
**Solution:**
- Check that css/ and js/ folders are uploaded
- Verify file paths in index.html
- Open browser console (F12) to check for errors
